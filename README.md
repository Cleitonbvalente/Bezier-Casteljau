# 🎯 Curva de Bézier com Método de De Casteljau

## 📌 Sobre o Projeto

Este projeto implementa uma **Curva de Bézier cúbica** utilizando o **método de De Casteljau**, com interpolação linear (LERP), desenvolvido com **HTML, CSS e JavaScript** usando o **Canvas 2D**.

A aplicação é interativa: o usuário pode arrastar os pontos de controle e visualizar a curva sendo atualizada em tempo real.

---

## 🧠 Conceitos Aplicados

- Curvas de Bézier
- Método de De Casteljau
- Interpolação Linear (LERP)
- Computação Gráfica
- Canvas 2D
- Eventos de mouse (interatividade)

---

## ⚙️ Funcionalidades

- 🎯 4 pontos de controle manipuláveis
- 🖱️ Arrastar pontos diretamente no canvas
- 🔄 Atualização dinâmica da curva
- 📈 Construção da curva via algoritmo de De Casteljau
- 🔗 Exibição das linhas de controle (guides)
